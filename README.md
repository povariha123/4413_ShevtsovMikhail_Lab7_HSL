# Практическая работа №7 — RGB ↔ HSL 

Программа для загрузки изображения, преобразования RGB → HSL, интерактивной коррекции H/S/L и сохранения результата обратно в RGB.

## Возможности ✨

- загрузка изображений PNG, JPEG, BMP;
- ручная реализация RGB → HSL и HSL → RGB;
- три ползунка для изменения Hue, Saturation, Lightness;
- отображение результата в реальном времени;
- цветной индикатор текущего Hue рядом с ползунком H;
- сохранение результата в PNG, JPEG или BMP.

## Запуск 🚀

```bash
pip install -r requirements.txt
python main.py
```
